# Features :
1. Retriving PDB file from PDB website.
2. Reading the retrived fasta file.
3. Acquiring cartesian co-ordinates from the file.
4. calculating torsion angle from the coordinates
5. GUI
6. Plotting of Ramachandran plot

# Libraries used :
1. Numpy
2. matplotlib
3. Tkinter
4. urllib
